Installation + configuration d'un container mmdebstrap bullseye

Lors de ma 2ème année j'ai installé et configuré un container sur DEBIAN 11 avec l'aide de mon professeur
M.LEMEUR. J'ai ensuite créé une procédure qui est jointe ci-dessous !

Dans ce container j'ai installer un serveur SSH(open-ssh), WEB(apache2) et plusieurs paquets nécéssaire à son
bon fonctionnement. J'ai également créé des utilisateurs et des groupes auquels j'ai attribué des droits
particuliers pour pouvoir lire/écrire/éxécuter des fichiers que dans certains dossiers.

Pour que tout le monde puisse accéder à mon container via le serveur SSH j'ai renseigné l'IP (10.0.31.171/8)
de celui-ci dans un tableau sur gitlab (via gitbash en clonnant par ssh avec une paire de clé) et créé un
compte utilisateur commun (alibo) avec lequel toute ma classe pouvait se connecter. Sur la page web par
défault de l'utilisateur alibo (10.0.31.171/~alibo/) j'ai aussi décris comment modifier sa propre page web
sur mon container.
